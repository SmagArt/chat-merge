# Merge Chat v2.4

Объединяет переписки **Telegram**, **ВКонтакте**, **Instagram** и **WhatsApp** в один файл TXT или Markdown.  
Расшифровывает голосовые сообщения офлайн через OpenAI Whisper.  
Поддерживает GPU-ускорение (NVIDIA CUDA, Apple Silicon MPS).

---

## Возможности

- Парсинг Telegram (JSON и HTML), ВКонтакте (HTML), Instagram (JSON), WhatsApp (TXT)
- Объединение нескольких папок одного контакта
- Расшифровка голосовых сообщений и кружочков через Whisper (офлайн)
- GPU-ускорение: NVIDIA CUDA, Apple Silicon MPS, AMD/Intel DirectML
- Фильтр по датам
- Форматы вывода: TXT и Markdown
- Метки времени: включить / выключить
- Разбивка на файлы: один / по месяцам / по годам
- Кэш расшифровок — повторный запуск мгновенный
- Drag & Drop папки прямо в окно программы
- История 5 последних папок

## Установка

### Windows

**Вариант 1 — установщик (рекомендуется):**
1. Скачай `MergeChat_Setup_v2.1.1.exe` из [Releases](https://github.com/SmagArt/chat-merge/releases)
2. Запусти и следуй инструкциям
3. Python и все пакеты установятся автоматически
4. При первом запуске автоматически определится и установится GPU-версия PyTorch

**Вариант 2 — если Python уже установлен:**
1. Скачай архив с исходниками
2. Запусти `setup_base.bat`

### macOS

**Вариант 1 — готовый DMG (рекомендуется):**
1. Скачай `MergeChat_v2.1.1.dmg` из [Releases](https://github.com/SmagArt/chat-merge/releases)
2. Открой DMG, перетащи MergeChat в папку Программы
3. При первом запуске: правый клик → Открыть (обход Gatekeeper)

**Вариант 2 — сборка из исходников:**
```bash
git clone https://github.com/SmagArt/chat-merge.git
cd chat-merge
bash build_mac.command
```

## Использование

1. Запусти Merge Chat
2. Укажи папку с экспортом переписки (или перетащи папку в окно)
3. Введи своё имя (как оно записано в мессенджере)
4. Выбери модель Whisper для расшифровки голосовых:
   - `tiny` — быстро, менее точно
   - `small` — баланс скорости и качества (по умолчанию)
   - `medium` — хорошее качество (рекомендуется)
   - `large` — максимальное качество, медленнее
5. При желании задай фильтр по датам или разбивку файла
6. Нажми «Запустить»

После завершения нажми «Открыть папку» — файл будет выделен в проводнике.

### Скорость расшифровки

На NVIDIA RTX 3060 Ti: ~241 голосовое (модель `medium`) за ~8 минут.  
На CPU: в 5–10 раз медленнее. Рекомендуется GPU.

## Как экспортировать переписку

**Telegram:**  
Настройки → Экспорт данных Telegram → выбери нужные чаты → формат JSON

**ВКонтакте:**  
vk.com/data_protection → Запросить данные → Переписки → скачай архив

**Instagram:**  
Настройки → Центр аккаунтов → Ваша информация → Скачать информацию → Сообщения → Формат JSON

**WhatsApp:**  
Открой чат → имя контакта → Экспортировать чат → выбрать «Без медиафайлов» или «Включить медиафайлы» → сохранить ZIP-архив.  
Распакуй архив — в папке будет `_chat.txt` и медиафайлы (голосовые — `.opus`).
